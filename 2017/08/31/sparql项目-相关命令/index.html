ssh 实现端口映射

ssh实现从本地端口到远程端口的映射，通常情况下使用 -L 或者 -R参数，例如：ssh xxx@xxx.xxx -L 本机端口号:RemoteIP:远程主机端口号

-L 将本地的某个端口映射到远程主机的某个端口上，上例中就是将本地的8089端口映射到远程主机的8088号端口上，这样就可以直接telnet本机的8089端口来访问远程主机了。RemoteIP可以是127.0.0.1，此时127.0.0.1指的是远程计算机，而非本机地址。

-R 则正好与-L相反，它将远程主机的某个端口映射到本地的某个端口上，例如：ssh xxx@xxx.xxx -R 远程主机端口号:LocalIP:本机端口号

上传Hadoop文件

  1. 启动hadoop start-all.sh
  2. 查看文件 hdfs dfs -ls
  3. 建立目录 hdfs dfs -mkdir /user/ikse/xuqiang
  4. 上传文件 hdfs dfs -put /local/file /hdfs/dir
  5. 删除文件 hdfs dfs -rm /hdfs/file
  6. 删除目录 hdfs dfs -rm -r /hdfs/dir
